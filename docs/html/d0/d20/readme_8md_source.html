3 # Auto-Generate Documentation by Doxygen using the following steps
4 
5 1. Generate the deconfig file by the command $ doxygen -g deconfig
6 2. Open the deconfig file and source the required directory. It is pre-assigned to current working directory.
7 3. Setup the require working output path under OUTPUT_DIRECTORY.
8 4. Set the parameters INLINE_SOURCES, SOURCE_BROWSER and RECURSIVE to YES
9 5. Run the command $ doxygen deconfig
