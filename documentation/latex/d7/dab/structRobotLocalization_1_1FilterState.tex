Structure used for storing and comparing filter states. 

This structure is useful when higher-\/level classes need to remember filter history. \mbox{\hyperlink{structRobotLocalization_1_1Measurement}{Measurement}} units are assumed to be in meters and radians. Times are real-\/valued and measured in seconds.
